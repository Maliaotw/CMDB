CMDB
===

# 說明

資產管理結合主機管理，IT人員可以對Windows主機的系統硬件資訊進行採集，並回傳給服務器，其他部門可以發起資產報修以及檢視資產維修紀錄，管理者可以管理用戶、部門、資產。

# 目錄結構

目錄結構分二個部分，[AutoCmdb]()負責資產管理網頁，[client]()負責採集主機硬件資訊到資料庫。
